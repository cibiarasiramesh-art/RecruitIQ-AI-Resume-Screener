Configure Google OAuth

Google Authentication is enabled but the credentials are missing or not loaded. Follow these steps to configure Google Login.

1 Create a Project in Google Cloud Console
Go to the Google Cloud Console, create or select a project, search for OAuth consent screen, configure it for external users, and publish it.
2 Generate OAuth 2.0 Credentials
Navigate to APIs & Services > Credentials. Click Create Credentials > OAuth client ID. Select Web Application as the application type.
3 Configure Authorized Redirect URIs
In the client ID settings, add the following URL under Authorized redirect URIs:
{{ redirect_uri }}
4 Set up local environment variables
Create a file named .env in your project root directory and add the Client ID and Client Secret generated by Google:
SECRET_KEY=replace_with_secure_random PORT=5000 GOOGLE_CLIENT_ID=your-google-client-id GOOGLE_CLIENT_SECRET=your-google-client-secret GOOGLE_REDIRECT_URI={{ redirect_uri }}
